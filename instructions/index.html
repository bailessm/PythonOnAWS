Running Python on AWS

Learning Objectives

  • Learn the basic python syntax
  • Learn how to use python to interact with AWS services
  • Learn how to run your python scripts on AWS without servers
  • Learn how to consistently deploy your python application to AWS
  • Learn how to invoke you python scripts with different event types

Introduction

This workshop teaches you how to use python with AWS services. You will then learn how to host your python scripts on AWS so that they can be invoked with common event patterns.

There are two parts to this workshop. Part 1 is learning the python syntax and service integrations using the Learn Python on AWS Workshop. This is completed in Task 2 and can take 4 to 8 hours to complete. This does not have to be completed all at once.

After you are comfortable with the basics of python and how you use it with AWS services, you return and pick up at Task 3.

In Task 3 you will learn how to use AWS Lambda to run python code on AWS. You will use both the AWS console and cli to create, deploy, and invoke Lambda functions.

Once you are comfortable with Lambda you will learn how to use the AWS Serverless Application Model (SAM) for deploying your Lambda functions. Using SAM you will add other services to invoke the Lambda functions with different types of events.

You will finally be given a challenge to develop your own application. The proposed application will use Amazon S3, Lambda, and DynamoDB, but at this point you should be equipped to develop anything using python to tie AWS services together.
